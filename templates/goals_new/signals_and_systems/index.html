Complex Numbers

Complex numbers are fundamental in Signals & Systems. They are used to represent signals, phasors, and analyze system behavior. A complex number can be expressed in two common forms: rectangular (x + jy) and polar (r∠θ). Understanding how to convert between these forms is essential for signal analysis and system design.

Definition & Components

A complex number has a real part and an imaginary part.

z = x + jy

  • x: Real part
  • y: Imaginary part

Conjugate

z* = x - jy

Magnitude & Phase

|z| = √(x² + y²), θ = arctan(y / x)

Operations

  • Addition: (x1+jy1)+(x2+jy2) = (x1+x2)+j(y1+y2)
  • Subtraction: (x1+jy1)-(x2+jy2) = (x1-x2)+j(y1-y2)
  • Multiplication: (r1∠θ1)*(r2∠θ2) = r1*r2 ∠ (θ1+θ2)
  • Division: (r1∠θ1)/(r2∠θ2) = (r1/r2) ∠ (θ1-θ2)

Euler's Formula

e^(jθ) = cosθ + j sinθ → z = r e^(jθ)

Polar & Rectangular Forms

A complex number can be represented in rectangular form as z = x + jy or in polar form as z = r ∠ θ, where:

  • r = magnitude = distance from origin = √(x² + y²)
  • θ = phase angle = angle with the positive x-axis

Conversion Formulas

  • Rectangular → Polar: r = √(x² + y²), θ = arctan(x / y)
  • Polar → Rectangular: x = r cosθ, y = r sinθ

Quadrant Check & Angle Adjustment

Since arctan only returns values between -90° and 90°, you must check the signs of x and y to determine the correct quadrant:

  • Quadrant I (Q1, ++): x>0, y>0 → θ = arctan(x/y) → 0°–90°
  • Quadrant II (Q2, -+): x<0, y>0 → θ = 180° + arctan(x/y) → 90°–180°
  • Quadrant III (Q3, --): x<0, y<0 → θ = 180° - arctan(x/y) → 180°–270°
  • Quadrant IV (Q4, +-): x>0, y<0 → θ = 360° + arctan(x/y) → 270°–360°
  • On y-axis (x=0): θ = 90° if y>0, θ = 270° if y<0
  • On x-axis (y=0): θ = 0° if x>0, θ = 180° if x<0

Tips for Using Quadrants

  • Check the signs of x and y first before applying arctan.
  • Always adjust θ to get a value between 0° and 360° for polar representation.
  • Remember: magnitude r is always positive; the sign is in the angle θ.
  • Use θ in radians if your calculations are for MATLAB, Python, or other programming tools.

Examples

  • Example 1: z = 3 + 4j r = √(3² + 4²) = 5, θ = arctan(3/4) ≈ 36.87° → Quadrant I z = 5 ∠ 36.87°
  • Example 2: z = -3 + 4j r = √((-3)² + 4²) = 5, θ = arctan(-3/4) ≈ -36.87° → Quadrant II → θ = 180° - 36.87° = 143.13° z = 5 ∠ 143.13°
  • Example 3: z = -3 - 4j r = √((-3)² + (-4)²) = 5, θ = arctan(-3/-4) ≈ 36.87° → Quadrant III → θ = 180° + 36.87° = 216.87° z = 5 ∠ 216.87°
  • Example 4: z = 3 - 4j r = √(3² + (-4)²) = 5, θ = arctan(3/-4) ≈ -36.87° → Quadrant IV → θ = 360° - 36.87° = 323.13° z = 5 ∠ 323.13°

Why Polar Form is Useful

  • Simplifies multiplication and division of complex numbers: just multiply/divide magnitudes and add/subtract angles.
  • Essential in phasor analysis for AC circuits.
  • Helps visualize signal magnitude and phase in the complex plane.
  • Useful in Fourier and Laplace transforms for signals & systems analysis.
